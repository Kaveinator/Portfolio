Boombox Update The update that brought a new map called "Rising School"
Home / Projects / Tanki X / Boombox Update

Details

In Tanki X, there's a map called 'Old School' which is based on a map from its predecessor, Tanki Online, known as 'Boombox'. The remaster, immerses players in the a wet atmosphere with a thunderstorm, accompanied by ambient sounds of thunder.

In the attempt to make the map more vibrant and appealing, we made a new variant of the map called "Rising School". Instead of a gloomy atmosphere, players are met with the crack of dawn, symbolizing a new beginning. The enhancements weren't merely cosmetic, with the dissipation of the fog, there were new tactical opportunities, particularly favoring weapons with longer range.

Technologies UsedWhat engines, frameworks and tools were used

Unity Engine (Unity3D) Since the game is based on Unity Game Engine, the new map had to be made in Unity. Most of the tools were already built from the Madness Project. So this project didn't require much scripting

I used Unity to ..

  • .. adjust environment lighting
  • .. make the fog less dense
  • .. change map resources, for skybox and battle sounds
